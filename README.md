# Nabra — Discord Bot (learning project)

Discord bot project to learn Discord.js, Node.js, and simple audio/command handling.

## Goals

- Learn how to build and run a Discord bot with Discord.js
- Explore command handling, event listeners, and a simple music/player implementation

## Resources

- Discord.js guide: https://discordjs.guide/
- Discord developer docs: https://discord.com/developers/docs/intro
